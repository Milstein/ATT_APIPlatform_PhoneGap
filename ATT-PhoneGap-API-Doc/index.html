AT&T API Platform Plugin for Adobe® PhoneGap®


The AT&T API Platform plugin for Adobe PhoneGap (plugin) is designed to provide an easy way to develop PhoneGap mobile applications that can access the the AT&T API Platform and RESTful APIs using JavaScript and HTML5. The plugin leverages the AT&T network-based services to create feature-rich apps. Access to services such as SMS, Terminal Location, and MMS, in conjunction with the ability to enable direct carrier billing, provides a multitude of options not normally available to mobile application developers.
